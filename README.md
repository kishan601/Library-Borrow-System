# Library Borrow System

Welcome to the Library Borrow System, the thrilling saga where books are not immortal, copies are finite, and disappointment is fully automated.

## What this does

- Lists books and their available copies
- Lets users borrow a book like responsible adults pretending they will return it
- Lets users return books, which is rare but spiritually beautiful
- Refuses borrowing when no copies remain, because reality still exists

## Tech stack

- Node.js
- React
- SQL

## How to use

1. Open the app
2. Browse books
3. Borrow one before someone else with faster fingers ruins your day
4. Return it later, ideally before the library sends a strongly worded look

## Warning

This system is powered by logic, optimism, and the ancient law of limited inventory.
